# Cloudflare deployment

This project is a Next.js app deployed to Cloudflare Workers through OpenNext.

## Local checks

```bash
npm install
npm run build
npm run build:cloudflare
```

For a local Worker preview:

```bash
copy .dev.vars.example .dev.vars
# Fill the Supabase values in .dev.vars
npm run preview:cloudflare
```

## Cloudflare settings

Use these build settings when connecting the GitHub repository:

- Build command: `npm run build:cloudflare`
- Deploy command, if available: `npm run deploy:cloudflare`
- Output is generated by OpenNext in `.open-next`

Optional environment variables:

- `NEXTAUTH_SECRET`
- `NEXTAUTH_URL`

The current Cloudflare-only build does not use Supabase or any external database.
Initial data is loaded from `public/data/*.json`; edits and Excel imports are stored in the browser localStorage.

Do not deploy this app as a static Pages export. The app is a Next.js app built by OpenNext and should run as a Cloudflare Worker.
