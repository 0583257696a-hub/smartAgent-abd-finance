# UI/UX SPEC

## Layout
- Sidebar ימני קבוע ברוחב 280px
- Header עליון עם חיפוש גלובלי
- Content Grid בכרטיסים
- סטטיסטיקות בצד שמאל/תחתון
- כפתור הוספה צף

## Dashboard
להציג:
- חיפוש מהיר
- פעולות נפוצות: פדיון, ניוד, תביעות, הלוואות, הצטרפות, הוראת קבע, שינוי מסלול, דמי ניהול
- כרטיסים אחרונים
- תבניות מייל אחרונות
- סטטיסטיקות: כמות מיילים, מפקחים, מעסיקים, תבניות

## Email Card
שדות:
- company
- category
- action
- email
- notes
- source

## Template Card
שדות:
- title
- category
- subject
- body

## Password Card
שדות:
- company
- system
- url
- username
- password
- notes

## Proportions
- אין כרטיסים גדולים מדי
- כל Card ברוחב 260-320px
- Grid: auto-fit minmax(280px, 1fr)
- Padding פנימי 20px
- Gap 18px
- כפתורים בגובה 40px
- חיפוש בגובה 58px

## לא לעשות
- לא להשתמש בטבלאות ענק כעיצוב ראשי
- לא להעמיס צבעים
- לא להקטין טקסטים מדי
- לא ליצור layout רחב מדי או ריק מדי
