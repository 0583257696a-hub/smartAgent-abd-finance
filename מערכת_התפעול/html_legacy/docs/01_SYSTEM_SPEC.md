# מערכת תפעול לסוכן ביטוח ופנסיה — SPEC מלא לקודקס

## מטרה
לבנות מערכת WEB פנימית בעברית מלאה RTL עבור סוכן ביטוח/פנסיה, שמרכזת את כל המידע התפעולי במקום אחד:
מיילים תפעוליים, תבניות מייל, מפקחים, מעסיקים, מוקדי שירות, מספרי סוכן, דמי ניהול, חשבונות להפקדה, הנחות ביטוח, קודי מוסד, קישורים, סיסמאות ומערכות.

המערכת מיועדת לעבודה יומיומית מהירה. היא לא אמורה להיראות כמו Excel ולא כמו מערכת ביטוח ישנה.

## טכנולוגיה
בשלב ראשון לבנות כקבצי Frontend בלבד:
- index.html
- style.css
- script.js
- data/*.json

ללא שרת וללא DB.
הנתונים נטענים מתוך JSON ונשמרים לעריכה מקומית ב-localStorage.
חובה לתמוך בייבוא/ייצוא JSON.
