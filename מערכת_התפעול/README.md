# מערכת התפעול

מבנה תיקיות מסודר:

- data/web - קבצי JSON של מערכת ה-WEB החדשה ב-Next.js.
- data/html - קבצי JSON של מערכת ה-HTML הישנה.
- html_legacy - עותק מסודר של גרסת ה-HTML הישנה לצפייה/השוואה.
- archive - מקום לשמירת עותקים ישנים בלי לערבב אותם עם המערכת הפעילה.

חשוב:
מערכת ה-WEB הפעילה נמצאת כרגע בשורש הריפו כדי ש-Vercel יבנה אותה ישירות.
לא להזיז את תיקיות src/public/supabase/prisma מהשורש בלי לעדכן את הגדרות Vercel.
